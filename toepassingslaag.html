Toepassingslaag

De toepassingen laag bestaat uit software, waaronder de toepassingen of applicaties vallen. Hierbij moet je denken aan apps en games die je kan downloaden op je telefoon.

Apps

Een toepassingsprogramma, ook wel 'app' genoemd, is een stukje software dat is ontworpen om specifieke taken uit te voeren op een computer, smartphone, of ander elektronisch apparaat. API

Een API (Application Programming Interface) is een software-interface die het mogelijk maakt dat twee applicaties met elkaar kunnen communiceren. Je zou kunnen zeggen de boodschapper die een verzoek indient bij een provider en vervolgens het antwoord terugkoppelt.
